    MajorDoMo LMS Connect Pugin

    Плагин для интеграции Logitech Media Server с системой домашней автоматизации MajorDoMo.

    Для установки плагина необходимо добавить ссылку http://skysilver-lab.github.io/MajorDoMo-LMS-Connect/repository/repo.xml в центр управления плагинами LMS.

    Проект поддерживается skysilver
