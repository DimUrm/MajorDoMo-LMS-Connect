    MajorDoMo Connect Pugin

    Плагин для интеграции Logitech Media Server с системой домашней автоматизации MajorDoMo.

    To install the plugin, add the repository URL http://skysilver-lab.github.io/MajorDoMo-LMS-Connect/repo.xml to your squeezebox plugin settings page.
